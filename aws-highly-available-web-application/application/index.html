AWS Highly Available Web Application

Three-Tier Architecture Deployment

Project Overview

This sample web application demonstrates a highly available three-tier architecture deployed on Amazon Web Services (AWS).

Networking

  • Amazon VPC
  • Public & Private Subnets
  • Security Groups

Compute

  • Amazon EC2
  • Application Load Balancer
  • Auto Scaling Group

Database

  • Amazon RDS (MySQL)
  • Multi-AZ Deployment

Monitoring

  • CloudWatch
  • SNS Notifications
  • Amazon S3
